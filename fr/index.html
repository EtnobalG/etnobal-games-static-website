Etnobal Games est un studio de jeu vidéo créé en 2021 par un mordu de jeu vidéo de la province de Québec au Canada. Pour le moment, notre expertise est centré sur le moteur de jeu Unity et le développement de jeu Web. Le studio dispose seulement d'une présence en ligne, nous ne disposons pas de bureau physique. Nous sommes présentement dans le développement de notre premier jeu avec Unity. Les informations sur celui-ci n'ont toujours pas été révélé.
